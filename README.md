# このプロジェクトについて
このプロジェクトはASP.NET MVC 4の機能であるASP.NET Web APIの使用方法に関するサンプルです。

## このプロジェクトの構成

Visual Studion 2012のソリューションディレクトリとして構成されています。

### SuperHero2
このディレクトリはサービス側の実装サンプルで、ASP.NET MVC 4のプロジェクトとして構成されています。

### SuperHero2Client
このディレクトリはサービスクライアントの実相サンプルで、Windowsのコンソールアプリケーションのプロジェクトとして構成されています。

##ビルド方法・ビルド環境

Visual Studio 2012 Ultimateにより開発されています。
ソリューション、プロジェクトとしてはUltimateとしての機能は使用していないので、Visual Studio 2012 Professional Edition以上であればビルドが可能です。
